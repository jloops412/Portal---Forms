Books and Book Pages

When selecting a book, a button for starting the Book Reader appears in the learning content overview. Depending on your settings, the book will be opened either in a new window or the same one. Close the Book Reader if you would like to return to the Library.

You can scroll through the book pages in the Book Reader as well as start and look at the objects filed in the book pages. Please refer here to the Help function in the Book Reader.

Displaying a book page

If a book page is not assigned to a book, then the page will be displayed, after it has been selected, directly in the learning content view in the course structure.

You can then hide the course structure if you only wish the book page to be displayed. The central bar can be used to show the course structure again so that additional learning content can be selected.

Opening modes and documents from a book page

As with the Library, learning content is also started in the corresponding programs. That means that when selecting a mode, the Trainer is started, whereas when selecting a document it is the corresponding program with which the document can be displayed that is opened.
