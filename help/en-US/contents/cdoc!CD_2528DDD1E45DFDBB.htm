Settings for Safari

The following settings are required for a flawless playback experience in Safari.

  1. Playback settings
    Open the browser's settings dialog using the Edit -> Preferences menu. In this menu, select the Security area and apply the following settings:

  2. Printing the stop page (OS X only)
    To print the stop page of the Trainer correctly you have to allow the print of background elements. Activate the option Print background in print dialog.
