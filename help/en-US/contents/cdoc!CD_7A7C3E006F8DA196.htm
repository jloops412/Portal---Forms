Trainer Bar Architecture and Components

The following functions are available to you in the Trainer bar. The components listed below therefore represent the Trainer's basic functionality.

  1. Orientation
    This function allows to position the Trainer bar optionally at the bottom of screen or at the top.

  2. Control
    These buttons allow the way a simulation unfolds to be individually controlled.

  3. Text
    This button enables you to set quickly if text bubbles shall be displayed or hidden.

  4. Audio
    This button enables you to switch quickly between audio playback and silent mode. It only appears in simulations with audio enabled contents.

  5. Volume
    This function allows you to control or deactivate audio file volume and bubble effects. Clicking on the button opens a selection menu from which the volume can be controlled and the sound deactivated by means of a selection field. It only appears in simulations with audio enabled contents.

  6. Progress display
    The progress display provides an overview of the steps and displays how the simulation is progressing. Among other information, it displays the title of the chosen simulation and name of the step currently being presented.

  7. Help
    This button displays the help of Trainer functionality.

  8. Close
    The Close button terminates the mode and the Trainer is closed.
