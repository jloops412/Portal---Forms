Selecting a Book Page Object

A book page can contain different objects. These objects are opened in different ways in the appropriate programs.

Modes

Modes cause buttons to be displayed which allow the respective mode to be invoked. This mode is then launched in the Trainer. Terminating a mode returns you to the Book Reader.

Media files

A book page can present you with various media files for audio or video content.
Appropriate programs are required for playing back media files. Windows Media Player should be installed as standard on your Windows operating system.

Note:

In certain circumstances it may happen that a Windows Media Player film is not played back correctly if the graphics driver is not compatible with the installed media player. One way in which this manifests itself is if the sound is played back but no image is visible.
To be able to play the film back correctly, the video acceleration setting needs to be customized. This is done by right clicking on the object and selecting Options... in the context menu. Now move the Video acceleration slide control to the middle setting or to None and retest the playback. This allows you to ascertain the optimal setting for your graphics driver.

Documentation

You can call different documentation types via the book pages. For example, these are files in Word or PowerPoint format as well as HTML pages or documents in PDF format. When a document is selected, it is opened directly in an associated application.
Please therefore bear in mind that the relevant application is required on your system to view the format in question. For example, you need a PDF reader to display PDFs or Microsoft Office to display the appropriate formats.
